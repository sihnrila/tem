# Cafe24 Demo Pages

카페24 판매용 페이지를 원본 스타일에 맞춰 퍼블리싱한 정적 데모입니다.

## 포함 내용
- 메인 리스트 페이지 (`index.html`)
- S128 상세 페이지 (`product/s128-detail.html`)
- Sage 데모 페이지 (`demo/sage/index.html`)
- 공통 스타일 (`css/style.css`, `css/detail.css`, `demo/sage/style.css`)

## 실행 방법
```bash
python3 -m http.server 8080
```

브라우저에서 `http://localhost:8080` 으로 접속하세요.

## 주요 경로
- 메인: `/index.html`
- 상세: `/product/s128-detail.html`
- 데모: `/demo/sage/index.html`
